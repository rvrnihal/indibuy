## 🚀 Quick Start Guide

### 📋 Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- MongoDB/MySQL database
- Git

### ⚡ Installation

```bash
# 🔄 Clone the repository
git clone https://github.com/yourusername/IndiBuy.git
cd IndiBuy

# 📦 Install dependencies
npm install

# 🔧 Environment setup
cp .env.example .env
# Edit .env with your configuration

# 🗄️ Database setup
npm run db:migrate
npm run db:seed

# 🚀 Launch development server
npm run dev
```
